🚀 HOW TO DEPLOY (5 STEPS)
==========================

1️⃣ SET UP SUPABASE (5 mins)
   - Create account at supabase.com
   - Create new project
   - Get API keys
   - Create database table (SQL provided)

2️⃣ CONFIGURE LOCAL (3 mins)
   - Add Supabase keys to .env.local
   - Run: npm run dev
   - Test locally ✅

3️⃣ PUSH TO GITHUB (4 mins)
   - Initialize Git
   - Create GitHub repo
   - Push your code

4️⃣ DEPLOY TO VERCEL (8 mins)
   - Connect Vercel to GitHub
   - Add environment variables
   - Click Deploy
   - Get live URL! 🎉

5️⃣ SHARE (1 min)
   - Send URL to your partner
   - Create memories together!
